2- Qual necessidade influenciou a criação do HTML?

4- Por que o HTML venceu a disputa entre as linguagens de marcação Web?

R: HTML é o que configura a forma como uma página Web e explica e renderiza pelo navegador ele é utilizado para marcar o conteúdo de uma página Web, e fornece estruturas semanticais do conteúdo permitindo que o navegador exiba a página.
