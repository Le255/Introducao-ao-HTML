Evolução do HTML


O HTML é uma linguagem padrão para a criação de páginas na web, aonde ele define o layout usando diversas tag que descrevem o conteúdo da página como texto, imagens, link e outros elementos. Sendo desenvolvido por Tim Berners. A primeira versão do HTML foi desenvolvida em 1993 sendo conhecida como HTML 1.0, visando o compartilhamento de informações; por falta de desenvolvedores a linguagem teve pouca evolução. Em 1995 o HTML 2.0 foi disponibilizada conforme a web ganhava popularidade, e foi nesse momento que a W3C foi formada. O projeto HTML 3.0 acabou não ser levado adiante, em 1994 o W3C criou o HTML 3.2, se tornou um padrão oficial servindo de suporte para a CSS. Com grandes mudanças veio o HTML 4.0 versão que representou um grande avanço na linguagem; em 1999 tivemos o HTML 4.1 onde foram incluíso várias tag. A última grande revisão foi o HTML 5.0 trazendo grandes mudanças, em 2014 o HTML 5.0 tornou-se uma recomendação, o mesmo aconteceu com o HTML 5.2 foi lançado pelo W3C a qual prevalece até hoje.
